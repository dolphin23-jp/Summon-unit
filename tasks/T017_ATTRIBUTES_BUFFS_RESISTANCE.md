# Task 017: 属性・バフデバフ・状態耐性

## 目的

7属性の相性、主要戦闘ステータスのバフ・デバフ、タグ式状態耐性を決定論的な戦闘計算へ追加する。

## 背景・参照仕様

- `docs/05_MONSTER_SYSTEM.md`
- `docs/07_BATTLE_EFFECTS.md`
- `docs/15_BALANCE_GUIDELINES.md`
- `docs/16_DATA_SCHEMAS.md`
- `docs/17_IMPLEMENTATION_ROADMAP.md`
- 前提タスク: T016

## 実装範囲

- 無・火・水・風・地・光・闇の属性IDと相性表
- 有利1.25、不利0.80、等倍1.00の整数倍率
- 属性倍率を単体ダメージ計算とAI予測へ統合
- Attack / Defense / Speed / Action Cost / Healing Receivedのバフ・デバフ
- `ActiveEffectState`を入力とするステータス倍率集計と上限処理
- 状態耐性タグと無効 / 50 / 75 / 100 / 125の5段階
- 状態耐性倍率の決定論的解決API

## 対象外

- 毒・火傷など個別状態異常の付与・継続処理
- 回復技そのものの実装
- 属性・状態アイコンのUI表示
- 確率命中・確率耐性

## 守るべき不変条件

- すべての倍率は整数permilleで保持する
- 属性相性とステータス上限は一つの定義へ集約する
- 同じ効果集合から同じ補正値を得る
- AI予測ダメージと実際のダメージ計算を一致させる
- 状態耐性は乱数を使わずタグと段階だけで解決する
- 既存の無効果戦闘結果は、属性相性を除き不必要に変えない
