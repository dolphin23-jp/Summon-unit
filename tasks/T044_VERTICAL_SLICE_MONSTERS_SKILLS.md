# T044 スライスモンスター・技一式

## 目的

垂直スライスで使用するR1～R5の16モンスター、固有特性、固有技、開花技、汎用技10種を固定IDのマスターデータとして確定し、T045の研究網・地域A/B制作へ渡せる状態にします。

## 完了条件

- [x] R1を6体実装する
- [x] R2を4体、R3を3体実装する
- [x] R4を2体、R5を1体実装する
- [x] 全16体へ一体一文票と必須設計項目を付ける
- [x] 全16体へ固有特性、固有技、開花技を一つずつ定義する
- [x] 汎用技T1を5種、T2を5種定義する
- [x] 汎用技効率を専用技比70～90%の帯へ固定する
- [x] 全開花技へ解析度・通貨・研究データ費を定義する
- [x] 件数、レア度配分、ID参照、一体一文票、100回決定論テストを追加する
- [x] ブラウザ起動時と`pnpm validate:content`でT044マスターも検証する
- [x] README、authoring rules、STATUSを更新する

## 実装

`src/content/vertical-slice-content.ts`へ次を集約します。

- `VERTICAL_SLICE_MONSTER_RECORDS`: 16体の戦闘マスターと一体一文票
- `VERTICAL_SLICE_PASSIVE_TRAITS`: 16固有特性
- `VERTICAL_SLICE_SKILL_RECORDS`: 固有16、開花16、汎用10
- `VERTICAL_SLICE_GENERIC_SKILL_COSTS`: 汎用技の個体育成費
- `VERTICAL_SLICE_BLOOM_RESEARCH_DEFINITIONS`: 種単位の開花研究費
- `VERTICAL_SLICE_CONTENT_CATALOG`: T045へ渡す独立カタログ

各モンスター票は名称、役割一文、主役割・副役割、推奨配置、長所、弱点、相性のよい味方、敵側対策、検証要素を保持します。技は表示名、戦術説明、tier、効果意味タグ、既存`SkillDefinition`を保持します。

## 設計判断

T044では既存T039デモカタログを置き換えず、垂直スライスマスターを独立して追加します。これにより現在のセーブ・デモ互換を維持し、T045で研究網、地域、ステージ、初期PlayerDataを一度に切り替えられます。

効果意味は固定`effectTags`で記録し、モンスター名や技名によるエンジン分岐は追加しません。T045の戦闘・研究配線は固定IDと効果部品を参照します。

数値はT046のシミュレーション調整前の初期値です。T044ではレア度差、役割差、行動コスト帯、汎用技効率帯を確定し、勝率調整は行いません。
