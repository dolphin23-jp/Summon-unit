# T050 — 初回導線・用語ヘルプ・確認統一

## 目的

初見プレイヤーが次に行う操作を判断でき、ゲーム固有用語を画面内で確認でき、進行を失う可能性がある操作には一貫した確認が入る状態にする。

## 実装範囲

- 未クリア時のホームに「最初の3手」を表示する。
- ホームおよび主要画面から開ける用語ヘルプを追加する。
- セーブ管理の上書き、ロード、削除、JSONインポートを共通確認ダイアログへ統一する。
- 確認文は「何が起きるか」「取り消せるか」「実行対象」を明示する。
- キーボード操作とdialog semanticsを保つ。

## 対象外

- レイアウト全面改修
- 戦闘演出
- 多言語対応

## 完了条件

- [ ] 初期進行で推奨順序が表示される
- [ ] 用語集へ主要画面から到達できる
- [ ] 危険操作が共通ダイアログを経由する
- [ ] キャンセル時に操作が実行されない
- [ ] test / content validation / benchmark / lint / buildが通る
