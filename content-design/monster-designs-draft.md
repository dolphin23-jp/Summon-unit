# Monster Designs — Vertical Slice 叩き台（16体）

`docs/14_VERTICAL_SLICE.md` の初期モンスター案（名称・役割は仮）を、`docs/13_CONTENT_AUTHORING_RULES.md` の必須設計項目に沿って第一稿としてすべて埋めたものです。数値は`docs/15_BALANCE_GUIDELINES.md`の式・目安に沿った概算で、確定値ではありません。

## 凡例・前提

- ステータス基準個体（docs15）: HP120 / ATK30 / DEF20 / SPD3。ダメージ = `attack * skillMultiplier * 100/(100+defense)`
- 行動コスト帯（docs03）: 非常に軽い60-70 / 軽い75-85 / 標準100 / 重い130-150 / 大技170-200 / 通常移動約75
- 到達方式はdocs04のDeliveryMode（CONTACT/DIRECT/PIERCE/ARC/SNIPE/SELF/GLOBAL）。ただし味方対象の支援技は遮蔽の対象外（「味方は味方の攻撃を遮らない」）のため、幾何区分を強制せず「対象: 味方単体」等で表記
- IDはドラフト用の仮スラッグ（`species.xxx` / `skill.xxx`）。正式命名はT044で決定
- レア度ごとの検証概念はdocs13準拠: R1は概念1つのみ、R2は専門性、R3は複数要素の組み合わせ、R4は編成中核、R5は地域到達点

## R1（概念1つのみ）

### 1. 草牙ネズミ — 高速

**一文役割**: 低耐久ながら行動間隔が短く、先手で数を稼ぐ近接アタッカー。

| 項目 | 内容 |
|---|---|
| ID | species.kusakiba-nezumi |
| 属性 / 種族 | 無 / 獣 |
| タグ | なし |
| 主役割 / 副役割 | 速攻アタッカー / 先制の数的圧力 |
| 推奨配置 | 前列 |
| ステータス | HP85 / ATK28 / DEF12 / SPD5 |

- 固有特性: 「疾走」— 通常移動の行動コスト-15%
- 固有技: 「引っ掻き」CONTACT / cost70 / power0.7
- 開花技: 「加速の一歩」— 使用後、自身の次回行動コストを-20%（1回のみ有効）

- 長所: 行動回数が多く先制しやすい
- 弱点: HP/DEFが低く、重技一発で落ちる
- 相性: 耐久型の前衛やバフ役と組むと数を活かせる
- 対策: 範囲攻撃・貫通技、単体高火力
- 推奨AI: 強襲
- 研究元 / 研究先: 起点 / 疾風狼
- 検証要素: 行動順・行動コストの基本感覚
