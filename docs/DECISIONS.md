# Design Decisions

## 確定事項

- 各陣営3×3、最大9体
- 同種・高レアの編成制限なし
- 編成コスト上限なし
- コストは経済のみ
- ラウンド制ではなく次回行動時刻
- 技または移動を1行動として選ぶ
- 固有・汎用・開花の最大3技
- 開花は早期・気軽に解放
- R1～R10、高レアは明確に強い
- 超高レアほど種類数が少ない
- レベル・個体値なし
- 研究は融合・系譜・収束・超越
- 設計図は永久、同種を無制限召喚
- オート基本、必要な一手だけ手動
- 通常は決定論、通常攻撃必中
- ガチャ・スタミナ・現実時間待機なし
- IndexedDBとJSONエクスポート
- 固定ID・schema migration

## 明示的に非採用

- 変異研究（特殊戦闘条件）
- 編成総コスト上限
- 同種1体制限
- 高レアの強さをコストで完全相殺
- 初期の混乱・魅了・完全スタン
- 初期の複数マス大型ユニット
- 通常の命中・回避・会心乱数
- 永久消滅
- 研究・召喚の確率失敗

## 暫定値

- ダメージ式
- 属性相性と倍率
- 基準ステータス
- 行動コスト帯
- 待機コスト
- 時間操作上限
- 召喚倍率
- 還元率
- 状態効果量
- 時間切れ判定

暫定値はテストで変更可能ですが、データとして差し替え、ルールをコードへ散在させません。
