# Glossary

- **種（Species）**: モンスター設計図・基本性能・解析度を共有する単位
- **個体（Instance）**: 実際に召喚され、汎用技習得等を持つ一体
- **固有特性**: 行動枠を使わない専用能力
- **固有技**: 変更不能の基本アクティブ技
- **汎用技**: 個体ごとに習得し一つ装備する自由枠
- **開花技**: 早期解放される種専用の第三技
- **解析度**: 種への理解・研究情報を開示する進行値。レベルではない
- **設計図**: 一度研究すれば永久に召喚可能になる解放情報
- **研究標本**: 確定研究で消費される召喚済み個体
- **行動コスト**: 次回行動までの基礎負担
- **次回行動時刻**: タイムライン上で次に行動する時刻
- **予約イベント**: 未来時刻に発動する予兆・召喚・設置物等
- **到達方式**: 接触・直線・貫通・曲射等、対象へ届く仕組み
- **効果範囲**: 着弾後に影響するマス
- **遮蔽**: 同列の前方敵が後方を直線攻撃から守ること
- **stable状態**: 一つの戦闘処理が完全終了し、安全に保存できる状態
- **垂直スライス**: 中核ループを最小コンテンツで通して検証する版
