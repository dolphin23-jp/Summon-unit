# 05. Monster System

## 基本ステータス

- HP
- Attack
- Defense
- Speed
- Attribute
- Rarity
- Primary Species
- Tags

レベル、個体値、ランダム成長はありません。

## 能力枠

- 固有特性: 常時・反応型の専用能力
- 固有技: 変更不能の基本行動
- 汎用技: 個体ごとに習得・1つ装備
- 開花技: 種単位で早期解放・1つ装備

## レア度

内部値はR1～R10です。

| レア度 | 設計上の位置づけ |
|---|---|
| R1 | 基本概念を一つ示す |
| R2 | 明確な専門性 |
| R3 | 複数要素の組み合わせ |
| R4 | 編成中核候補 |
| R5 | 地域段階の強力な到達点 |
| R6 | 通常種を大幅に上回る |
| R7 | 一系統の頂点 |
| R8 | 複数系統の統合 |
| R9 | 世界設定上の例外 |
| R10 | ルールを変える超越存在 |

高レアは明確に強くて構いません。編成上限コストや同種制限で相殺しません。代わりに種類数・研究条件・召喚費を重くします。

## 属性

暫定7属性:

- 無
- 火
- 水
- 風
- 地
- 光
- 闇

暫定倍率:

- 有利: 1.25
- 不利: 0.80
- 等倍: 1.00

四属性は `火→風→地→水→火`、光と闇は相互有利を候補とします。

## 種族

暫定:

- 獣、鳥、虫、植物、水棲、竜、亜人、機械、霊体、不死、魔族、神獣

## タグ

複数付与可能:

- 飛行、重装、巨大、毒性、再生、装甲、人工、王族、遠隔、魔法、召喚物など

飛行は単一の万能能力ではありません。接触無効、地面領域無効、移動拡張、遮蔽無視を個別に付与します。

## 種状態と個体状態

### 種単位

- 設計図
- 解析度
- 開花技
- 図鑑開示
- 研究情報

### 個体単位

- instanceId
- 習得済み汎用技
- お気に入り・保護
- ニックネーム
- 修復・遠征状態

基本性能は同種で共通です。
