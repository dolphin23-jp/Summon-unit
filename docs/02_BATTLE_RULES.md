# 02. Battle Rules

## 戦闘参加数

- 味方1～9体
- 敵1～9体を基本
- 同種制限なし
- 召喚物を含め、各陣営の盤面占有は9マス以内

## 行動選択

行動順が来たユニットは次のいずれかを選びます。

- 固有技
- 装備中の汎用技
- 装備中の開花技
- 通常移動
- 待機

## 勝敗

- 敵の戦闘可能ユニットが0で勝利
- 味方の戦闘可能ユニットが0で敗北
- 召喚物だけが残った場合の扱いは召喚物データで指定可能。標準では戦闘可能ユニットとして数える
- 通常戦には安全のため最大仮想時刻を設ける

時間切れ判定の最終式は未確定ですが、優先候補は次です。

1. 残存ユニット数
2. 残存HP割合合計
3. 与ダメージ合計
4. なお同値なら敗北または引き分け（モード別指定）

## 戦闘不能

HPが0以下になると戦闘不能です。

1. 戦闘不能防止を確認
2. 戦闘不能確定
3. 予約された通常行動を削除
4. 占有マスを空ける
5. 戦闘不能時効果を処理
6. 撃破者を記録

通常ユニットは盤面から消えます。死体や残骸を残す場合は設置物生成として実装します。

## 蘇生

標準ルール:

- 元のマスが空いていれば元のマス
- 埋まっていれば隣接空きマス
- 最大HPの30%で復帰
- バフ・デバフ・通常状態異常は解除
- 原則1戦闘1回
- 次回行動は通常の初動時間後

## 待機

正式な行動です。

暫定コスト:

- 1回目: 50
- 2回連続: 65
- 3回以上: 80

技または移動で連続待機回数をリセットします。

## 手動介入

- AUTO OFFで次の味方行動開始時に停止
- 「次の味方行動だけ手動」を用意
- 手動決定後に自動へ戻せる
- 敵は常にAIで行動

## 決定論

- 通常攻撃は必中
- 通常の状態付与は確率を使わない
- 同点選択は固定規則
- ランダム技のみ戦闘シードを使用
