# 04. Board and Targeting

## 座標

内部盤面は6行×3列です。

- 行1: 敵後列
- 行2: 敵中列
- 行3: 敵前列
- 行4: 味方前列
- 行5: 味方中列
- 行6: 味方後列
- 列: A / B / C

座標は表示順ではなく固定の構造体で管理します。

## 占有

- 通常ユニット・占有型設置物は1マス
- 1マスに占有要素は1つ
- 非占有領域は別レイヤー
- 初期実装では複数マスユニットなし

## 通常移動

- 自陣内の隣接8方向
- 空きマスのみ
- 敵陣侵入不可
- 交換不可
- 連鎖押し出しなし

## 敵との接触

敵前列と味方前列は同じ列のみ通常接触します。斜め前を攻撃するには、その射程マスクを持つ技が必要です。

## 技定義の4層

1. 使用条件
2. 到達範囲（選択可能マス）
3. 到達方式（射線・遮蔽）
4. 効果範囲（着弾後の対象）

## 到達方式

- `CONTACT`: 接触
- `DIRECT`: 直線、最前の敵で停止
- `PIERCE`: 列を貫通
- `ARC`: 遮蔽無視のマス指定
- `SNIPE`: 遮蔽無視のユニット指定
- `SELF`: 自己中心
- `GLOBAL`: 全域

## 遮蔽

- 各列で攻撃者側に最も近い敵が後方を守る
- 味方は味方の攻撃を遮らない
- `ARC`, `SNIPE`, `PIERCE` 等で対処可能
- 前衛の押し出し・撃破・列移動でも崩せる

## 効果範囲マスク

- 単体
- 横一列
- 縦一列
- 対象＋左右
- 対象＋後方
- 十字
- 周囲8
- チェイン
- 陣営全域

数値距離だけではなく、理解しやすいマスクライブラリを使います。

## 強制移動

- 押し出し、引き寄せ、横移動
- 移動先が空いている場合のみ成功
- 盤外・占有時は標準では失敗
- 連鎖押し出しなし
- 失敗時の衝突ダメージ等は技個別

## プレビュー

選択時に以下を表示します。

- 選択可能マス
- 遮蔽で止まるマス
- 実際の着弾点
- AoE
- 強制移動先
- 移動後に使用可能になる技・対象
