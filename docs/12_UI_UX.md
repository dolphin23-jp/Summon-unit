# 12. UI / UX

## 戦闘画面

PC・タブレット横を基準とします。

- 中央: 6×3盤面
- 左: 選択ユニット詳細
- 右または上: タイムライン
- 下: ログ・手動コマンド
- 上: AUTO・一時停止・速度・ステージ情報

スマートフォンではパネルを展開式にします。

## 盤面の常時情報

- モンスター画像
- HP
- 障壁
- 主要状態最大4個
- 属性アイコン
- 次回行動簡易表示
- 予兆対象

レア度・全ステータス・全技を常時重ねません。

## タイムライン

未来10～12件:

- ユニット行動
- 予兆着弾
- 召喚
- 設置物
- フェーズ移行

技ホバー・選択時に使用後の次回位置を半透明で示します。

## 手動操作

- 固有技
- 汎用技
- 開花技
- 移動
- 待機

確定前プレビュー:

- 正確なダメージ・回復
- 撃破
- 遮蔽
- AoE
- 状態
- 強制移動
- 次回行動順位

決定論的な技は幅ではなく確定値を表示します。

## オート操作

- AUTO ON/OFF
- 一時停止
- 次の味方だけ手動
- ×1 / ×2 / ×4
- クリア済みで×8・即時シミュレーション候補

## 予兆

色だけに依存せず、枠線・模様・アイコンを併用します。

表示:

- 発動までの時間
- 属性
- 予測ダメージ
- 強制移動
- 中断条件

## 行動理由

通常ログは一行、詳細モードではスコア内訳を表示可能にします。

## 戦闘終了

- 報酬
- 修復費
- 解析度変化
- 開花・研究通知
- 個体戦績
- 再戦 / 編成 / 研究 / 次ステージ

## アクセシビリティ

- タッチだけで完結
- 色以外の識別
- `prefers-reduced-motion`
- 演出: 標準 / 簡略 / 最小
- キーボードは補助
