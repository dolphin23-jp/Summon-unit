# 01. Core Loop

## 基本循環

```text
ステージへ出撃
  ↓
決定論的オート戦闘（必要時のみ手動介入）
  ↓
通貨・研究データ・触媒・解析度を獲得
  ↓
研究ヒントやノードが開示
  ↓
試験研究で候補を絞る
  ↓
確定研究で設計図を永久解放
  ↓
通貨で必要数の個体を召喚
  ↓
配置・汎用技・開花技・AI方針を変更
  ↓
次のステージまたは再戦
```

## 一回のプレイで進むもの

毎回すべてを進める必要はありませんが、数戦して何も変わらない状態を避けます。

- 通貨または研究データ
- 解析度
- 開花研究
- 研究ヒント
- 触媒
- 新設計図
- 地域・施設段階

## 成長の表現

レベルやランダム個体値は使いません。

- 利用可能なモンスター種が増える
- 同種の所持数が増える
- 汎用技が増える
- 開花技によって運用が完成する
- 編成プリセットが洗練される
- 上位研究へ到達する

## 短期・中期・長期目標

### 短期

- 次の戦闘に勝つ
- 開花技を解放する
- 必要な触媒を得る

### 中期

- 地域ボスを倒す
- R4～R5の設計図を得る
- 同種複数編成を作る

### 長期

- 研究網を横断してR8以上へ到達する
- R9～R10を複数召喚する
- 深層・超越ボスへ挑む

## 周回方針

低確率ドロップだけに依存しません。

- 基本報酬は確定
- 希少素材はランダム追加可能
- 地域ポイント交換で天井を設ける
- クリア済み戦闘は高速化・即時シミュレーション可能
