# 00. Game Vision

## 目的

本作は、ブラウザで遊べる個人用のモンスター収集・研究・編成型タクティクスゲームです。

「新種を発見する楽しさ」と「配置・時間・技構成の結果を観察して改善する楽しさ」を中心に置きます。戦闘はオートを基本とし、必要な場面だけプレイヤーが一手介入できます。

## 中核となる感情

- 次の研究ノードの正体を知りたい
- 新しい設計図を解放したい
- 強力な個体をもう一体召喚したい
- 同じ敵に、配置や汎用技を変えて再挑戦したい
- 自分の編成が意図どおり動く様子を眺めたい

## 設計の柱

1. **研究による発見**: ガチャではなく、条件を満たすと確実に新種を得る
2. **決定論的な戦闘**: 同条件なら同結果となり、改善点を比較できる
3. **盤面と時間の両立**: 3×3配置と次回行動時刻の両方を考える
4. **理解できるオートAI**: 行動理由を確認でき、必要な一手だけ手動化できる
5. **明確な高レアの強さ**: 高レアは本当に強く、複数所持も許容する
6. **長期拡張性**: 100種類以上追加してもエンジンとセーブを作り直さない

## 非目標

- PvPの厳密な公平性
- すべてのモンスターを永遠に同格に保つこと
- ガチャ、スタミナ、デイリー、現実時間待機
- 低確率の命中・回避・通常会心に依存する戦闘
- 初期段階からの大量コンテンツ
- 原作ゲームの完全再現

## 成功基準

完成版の規模ではなく、垂直スライスで次が成立することを成功とします。

- 配置や技変更で結果が変わる
- オートを眺めるだけでも理解と発見がある
- 研究結果をすぐ編成で試したくなる
- 高レア解放に明確な達成感がある
- 新コンテンツをデータ追加中心で制作できる
